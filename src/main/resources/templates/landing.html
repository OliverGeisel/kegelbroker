Willkommen bei Kegelbroker

Kegelbroker ist ein System zum Sammeln bzw. Anzeigen von Kegelspielen in Echtzeit. Dies ist die Client-Anwendung. Diese Software läuft auf dem System, das die Kegelspiele durchführt oder direkt Zugriff auf die Daten hat. Es kann die Spiele direkt im Browser anzeigen und/oder die Daten an einen Server senden, der die Daten dann für andere Anwendungen bereitstellt oder anzeigt.
